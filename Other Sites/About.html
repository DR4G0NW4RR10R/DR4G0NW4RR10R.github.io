About

This is a website that I first started making in my 8th grade year, as a side project for a Digital Media and Literacy class I took. We were doing HTML, and I was bored, and thus, this website was made. Since then, my website gained popularity amongst my peers because it was offline and I had hacked some of the games myself, the games that everyone played, but wanted an easy way out. I periodically update my website, add more games, and a better UI, etc. Now, it has reached you, and I hope that you enjoy playing these games!

Happy Gaming!
